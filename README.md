# 🍕 Pizza Sales Analysis using SQL

## 📌 Project Overview
This project analyzes pizza sales data using **SQL** to extract meaningful business insights. The goal is to understand sales performance, customer preferences, and ordering patterns of a pizza restaurant.

The dataset includes information about orders, pizza types, quantities, prices, and timestamps. Using SQL queries, the project explores different aspects of the business to help identify trends and improve decision-making.

This project demonstrates practical SQL skills including **data aggregation, joins, grouping, filtering, and analytical queries**.

---

## 🗂️ Dataset Description

The dataset contains four main tables:

- **Orders**  
  Contains order ID, order date, and order time.

- **Order_Details**  
  Contains information about each order including pizza ID and quantity.

- **Pizzas**  
  Contains pizza size and price information.

- **Pizza_Types**  
  Contains pizza names, categories, and ingredients.

These tables are connected using **primary and foreign keys** to perform complete sales analysis.

---

## 🎯 Project Objectives

The main objectives of this project are:

- Analyze the **total number of orders and total revenue**
- Identify the **most popular pizza types**
- Understand **customer ordering behavior**
- Analyze pizza sales by **size and category**
- Discover **peak ordering hours**
- Determine **top-selling pizzas by revenue and quantity**

---

## 📊 SQL Analysis

### 1️⃣ Basic Analysis
- Total number of orders placed  
- Total revenue generated from pizza sales  
- Highest priced pizza  
- Most commonly ordered pizza size  
- Top 5 most ordered pizza types  

### 2️⃣ Intermediate Analysis
- Total quantity of pizzas sold by category  
- Distribution of orders by hour of the day  
- Category-wise pizza distribution  
- Average pizzas ordered per day  
- Top pizzas based on revenue  

### 3️⃣ Advanced Analysis
- Percentage contribution of each pizza type to total revenue  
- Cumulative revenue analysis over time  
- Top pizzas by revenue within each category
